HTML colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values

You can set your text color with the color attribute e.g

h1 style="color: green;"

You can set the background-color of your text with the same attribute e.g

h3 style="color: green;"

You can set the color of your border

h1 style="border: 2px solid tomato;"
